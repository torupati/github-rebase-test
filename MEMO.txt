Github repository で rebase したい

試してみた。

```
echo "# github-rebase-test" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:torupati/github-rebase-test.git
git push -u origin main
```

main update

```
$ git log --oneline --graph
* db94a3a (HEAD -> main, origin/main) main second
* bca9186 first commit
```

こういう感じでmain を進めています。
